Renesas Envision Kit RX65N ガジェット
=========
![R5F564ML](../RTK5RX65N.jpg)

## 概要
Envision Kit RX65N で実現するオーディオプレイヤー
   
## プロジェクト・リスト
 - main.cpp
 - Makefile
   
## ハードウェアーの準備
 - SD カードインターフェースの準備
 - オーディオの接続（DA0、DA1 から出力、GND レベルは、1.65V）
   
## リソースの準備
 - SD カードに、「wav, mp3」形式のファイルを書き込む。
   
## ビルド方法
 - make する。
 - audio_sample.mot ファイルを書き込む。
