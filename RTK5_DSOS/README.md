Renesas Envision Kit RX65N ガジェット
=========
![R5F564ML](../RTK5RX65N.jpg)

## 概要
Envision Kit RX65N で実現するデジタルストレージオシロスコープ
   
## プロジェクト・リスト
 - main.cpp
 - Makefile
   
## ハードウェアーの準備
 - SD カードインターフェースの準備
 - アナログ入力の接続
   
## リソースの準備
 - SD カード、ルートに「kfont.bin」ファイルを書き込む。（ファイラーでの漢字表示）
   
## ビルド方法
 - make する。
 - dsos_sample.mot ファイルを書き込む。
